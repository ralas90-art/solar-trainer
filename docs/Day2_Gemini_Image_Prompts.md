# Day 2 Presentation — Gemini Image Generation Guide

**How to use this guide:**
1. Open Gemini (gemini.google.com) and use **Imagen 3** image generation
2. For each image below, select the specified **aspect ratio** in the Gemini UI
3. Paste the prompt exactly as written
4. Save the output file using the exact **filename** listed (important — the injection script uses these names)
5. Save all images to: `C:\Users\12132\.gemini\antigravity\brain\d409138a-18b3-4f28-a396-07f832f2e8ca\`

---

## GLOBAL STYLE SEED
> Append this to the END of every single prompt below:

```
Photorealistic editorial photograph. Clean, modern, professional — no stock-photo cheese.
No text, logos, or watermarks inside the image. Lighting: bright natural daylight preferred;
avoid harsh flash. Diversity: represent a range of ethnicities, ages, genders.
Color palette: warm orange (#F4821F) and gold (#F5C842) accents acceptable but image should
NOT feel artificially tinted. Deep navy (#0D1F3C) backgrounds acceptable for scenario shots.
```
